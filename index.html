História dos computadores

Ante de começar a criar sites, é importante entender o passado sobre o que estamos estudando. Veja abaixo um breve resumo de onde tudo surgiu.

Primeira geração de computadores

A primeira geração de computadores foi marcada por computadores a válvula.

Harvard Mark I, ou Automatic Sequence Controlled Calculator, chamado popularmente de Mark I, foi um dos primeiros computadores eletromecânicos no Estados Unidos no ano de 1944.

Colossus, criado em 1946 pelos foi utilizado para criptoanálise pelos Britânicos.

ENIAC, ou Eletronic Numerical Integrator and Computer, era mais automático comparado ao Colossus e, ambos faziam cálculos relacionado á guerra.

Segunda geração (1959 a 1965)

A segunda geração de computadores teve como marca a chegada dos transistores. Permitindo que os computadores diminuíssem o seu tamanho.

Terceira geração (1965 a 1970)

O uso de transistores passou a ser substituído pelo de circuitos integrados, unidades de encapsulamento semicondutoras que agrupam transistores, resistores, diodos e outros componentes elétricos interligados em uma pastilha de Silício e Germânio, que passaram a ser conhecidos como chips.

Quarta geração (a partir de 1971)

A partir de quando começaram a aparecer os primeiros microprocessadores, tornando os computadores menores e portáteis.

Quinta geração

A quinta geração de computadores, conhecida como a geração da inteligência artificial, é composta por máquinas onde a capacidade de processamento de dados, taxas de transferência e armazenamento foram consideravelmente aumentadas. Desde o início desta geração e até hoje, ocorrem diversos processos de diminuição do espaço físico. Computadores cada vez menores são criados a cada dia.

O que são Clients?

Cliente é um termo empregado em computação e representa uma entidade que consome os serviços de uma outra entidade servidora, em geral através do uso de uma rede de computadores numa arquitetura cliente-servidor.

Características de Client

  • Inicia pedidos para servidores;
  • Espera por respostas;
  • Recebe respostas;
  • Conecta-se a um pequeno número de servidores de uma só vez;
  • Normalmente interage diretamente com os servidores através de seu software aplicação especifico, que lhe possibilita a comunicação com o servidor;
  • Utiliza recursos da rede.

O que são Servers?

Um servidor processa e executa solicitações feitas por usuário através de softwares, bancos de dados, envio e recebimento de informações como e-mails, envio de formulários, hospedagem de websites e outros.

Características de Servers

  • Sempre espera por um pedido de um cliente;
  • Atende os pedidos e, em seguida, responde aos clientes com os dados solicitados;
  • Podem se conectar com outros servidores para atender uma solicitação específica do cliente; jamais podem se comunicar.
  • Fornece recursos de rede.
  • Normalmente interage diretamente com os usuários finais através de qualquer interface com o usuário;
  • Estrutura o sistema.

Estrutura básica de um HTML

A estrutura básica de um documento HTML é composta por um conjunto de tags que definem o esqueleto do documento.

  • A tag HTML é o elemento raiz e contém todas as outras tags que compõem o elemento;
  • A tag head contém informações sobre a página, como título e metadados;
  • A tag body contém todo conteúdo visível na página.

Algumas tags não precisam de fechamento, elas são chamadas de tags vazias. Elas são usadas para incluir elementos que não possuem conteúdo, como por exemplo a tag <img> que é usada para incluir imagens em uma página HTML.

Diferença entre listas ordenadas e não ordenadas

Lista ordenada

Uma lista ordenada é delimitada pelas etiquetas <ol> e </ol> (Ordered List). Cada item da lista é delimitado pelas etiquetas <li> e </li> (List Item).

  1. Item número um.
  2. Item número dois.
  3. Item número três.

Lista não ordenada

A estrutura da lista não ordenada é semelhante a da lista ordenada - porém, ao invés de ser construída com o elemento ol, é construída com o elemento ul (Unordered List).

  • Primeiro item.
  • Segundo item.
  • Terceiro item.
